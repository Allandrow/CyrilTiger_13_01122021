# Projet #13 OpenClassrooms - Utilisez une API pour un compte utilisateur bancaire avec React

## Stack :

- Vite
- React
- Typescript
- More incoming

## Usage :

Download / clone this repository

Install Dependencies :

```bash
npm install
```

Launch dev server :

```bash
npm run dev
```

Build :

```bash
npm run build
```

Preview :

```bash
npm run serve
```
